# 小球大乱斗

一个单文件 HTML Canvas 小游戏。玩家控制小球在竞技场中移动、吞噬能量和更小的对手，通过分裂、吐球、冲刺、护盾等技能和 32 个 AI 对手竞争。

## 在线体验

项目可部署到 Vercel。首页提供了 GitHub 源码入口，线上玩家可以从游戏页面跳转回本仓库。

## 功能特点

- 单人竞技和团队模式
- 32 个 AI 对手
- 多种场景主题：霓虹、赛博、糖果、水墨、极光、晴空、宣纸、薄荷
- 多种小球皮肤
- 分裂、吐球、急停、再分、合球、冲刺、护盾、磁力、膨胀等操作
- 首页、结算页和游戏 HUD 均在同一个 `index.html` 中实现

## 本地运行

这个项目没有复杂依赖，直接在浏览器中打开 `index.html` 就可以运行。

也可以执行构建命令，把静态页面和 Worker 文件复制到 `dist` 目录：

```bash
npm run build
```

构建后输出：

- `dist/client/index.html`
- `dist/server/index.js`

## 操作说明

- `WASD`：移动
- `J`：分裂
- `K`：吐球
- `L`：急停
- `U`：再分
- `G`：合球
- `I`：冲刺
- `O`：护盾
- `H`：磁力
- `N`：膨胀

## 部署

仓库包含 `vercel.json`，Vercel 会执行：

```bash
npm run build
```

并使用 `dist/client` 作为输出目录。
